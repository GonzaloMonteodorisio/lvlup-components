# Get started

`lvlup-components` es una lib que implementa componentes comunes dentro del curso de LVL UP

Los componentes definidos deben recibir solamente objetos de dominio para establecer ...

![Summary Architecture Representation](foto.png)


## Install

### Con yarn

```bash
  yarn add @lvlup-components
```

> Nota: Esta libreria necesita como peer dependencies lvlup-js-utils & lvlup-hooks
